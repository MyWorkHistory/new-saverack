# CRM theme source

Color palette and radii follow **Vuexy** (`#7367F0` primary, light/dark surfaces) from the licensed Vuexy Vue.js admin template. This folder documents that lineage only.

Application styles are compiled from [`../styles/vuexy-entry.scss`](../styles/vuexy-entry.scss) (Bootstrap 5 + CRM layout). Full Vuexy/Vuetify source is **not** vendored in this repository—only design tokens and layout patterns are mirrored for Save Rack.
